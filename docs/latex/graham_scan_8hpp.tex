Return the Convex Hull (ordered in anticlockwise direction) of a set of points using the Graham\textquotesingle{}s Scan algorithm. Works in worse case time complexity O(\+Nlog\+N) and space complexity O(N+H) where N is the number of points and H is the number of points in the Convex Hull.
